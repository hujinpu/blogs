现在谈下我的想法：

  • 我觉得栅格系统其实是设计师多年设计经验的一个提到理论层次的结晶，所以它其实很强调实用性，固不可把它当个古董一样的玩意，纯属研究和收藏，它是可以很好的在实战中发挥威力的，特别是大型的网站。技术实现可以参考：
    1. Blueprint
    2. 960栅格系统
    3. YUI Grids CSS
  • 栅格系统用得好，其实不是很容易，有一定的计算量，需要大量的实践和总结。但用得好的化可以带来一下优势：
    • 能大大提高网页的规范性。在栅格系统下，页面中所有组件的尺寸都是有规律的。这对于大型网站的开发和维护来说，能节约不少成本。（开发起来有规可循）
    • 基于栅格进行设计，可以让整个网站各个页面的布局保持一致。这能增加页面的相似度，提升用户体验（用户是不愿意去时刻学习新东西的）。
