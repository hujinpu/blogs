对于Python列表，它是可变的，这点和字符串不同，所以你可以通过在等号的左边指定一个索引或者索引范围的方式来更新一个或几个元素，你也可以用 append() 方法来追加元素到列表中去

pop() 方法在删除列表中位置最后的一个元素的同时，也把这个元素作为返回值返回

remove(element) 方法会删除指定的元素，但只会删除一次，不会删除所有相同的element

再次强调：使用 extend() 方法比连接操作的一个优点是它实际上是把新列表添加到了原有的列表里面，而不是像 + 连接操作那样新建一个列表。

对于 * 重复操作符可以运用于所有序列对象，包括字符串、列表和元组

注意：list和tuple工厂函数都不是做的完全转换，而是新生成了一个对象，所以is操作符号会返回false

#!/usr/bin/env python

stack = []

def pushit():
    stack.append(raw_input('Enter new string: '))

def popit():
    if len(stack) == 0:
        print 'Cannot pop from an empty stack!'
    else:
        print 'Removed [', repr(stack.pop()), ']'

def viewstack():
    if stack:
        print str(stack)
    else:
        print 'empty stack!'

def showmenu():
    prompt = """
p(U)sh
p(O)p
(V)iew
(Q)uit

Enter choice: """

    done = 0
    while not done:

        chosen = 0
        while not chosen:
            try:
                choice = raw_input(prompt)[0]
            except (IndexError, EOFError, KeyboardInterrupt):
                choice = 'q'
            print '\nYou picked: [%s]' % choice

            choice = choice.lower()

            if choice not in 'uovq':
                print 'invalid option, try again'
            else:
                chosen = 1

        if choice == 'q': done = 1
        if choice == 'u': pushit()
        if choice == 'o': popit()
        if choice == 'v': viewstack()

if __name__ == '__main__':
    showmenu()

元组是不可变类型，所以可以用作字典的key，另外当处理一组对象时，这个组默认是元组类型，在赋值时如果左边只有一个变量，而右边是多个值，那么右边也会自动转为元组赋值给左边

有个关于列表和元组的工厂函数的细节要注意，就是当传入的参数是一个字符串时，会自动把字符串分解为一个个字符，这就意味着你最后得到的不是一个元素的列表或者元组，而是长度为字符串长度的列表或者元组

>>> str = 'abcdef'
>>> list(str)
['a', 'b', 'c', 'd', 'e', 'f']
>>> tuple(str)
('a', 'b', 'c', 'd', 'e', 'f')

虽然元组是不可变的类型，但是我们设法通过替换它的元素来“改变”它：

>>> t = (['xyz', 123], 23, -103.4)
>>> t
(['xyz', 123], 23, -103.40000000000001)
>>> t[0][1]
123
>>> id(t)
2146530484
>>> t[0][1] = ['abc', 'def']
>>> t
(['xyz', ['abc', 'def']], 23, -103.40000000000001)
>>> id(t)
2146530484

序列类型对象的浅拷贝是默认类型拷贝，并可以以下几种方式实施：

  1. 完全切片操作[:]
  2. 利用工厂函数，比如 list() ， dict() 等
  3. 使用copy模块的 copy 函数

对一个对象进行浅拷贝其实是新创建了一个类型跟原对象一样，其内容（可变类型的）是原来对象元素的引用，换句话说,这个拷贝的对象本身是新的，但是它的内容不是，但对于不可变类型的对象会显示的被复制，从而创建一个新的不可变的对象，不过对于整型和字符串都会有一定的缓存

>>> obj1 = ['hello world everybody', [1, 2, 3]]
>>> obj1
['hello world everybody', [1, 2, 3]]
>>> id(obj1)
2146529260
>>> id(obj1[0])
2146479232
>>> obj2 = list(obj1)
>>> obj2
['hello world everybody', [1, 2, 3]]
>>> id(obj2)
2146537708
>>> id(obj2[0])
2146479232
>>> obj2[1][0] = 'smellcode'
>>> obj2
['hello world everybody', ['smellcode', 2, 3]]
>>> obj1
['hello world everybody', ['smellcode', 2, 3]]

要得到一个完全拷贝或者说深拷贝--创建一个新的容器对象，包含原有对象元素（引用）全新拷贝的引用--需要 copy.deepcopy() 函数

>>> person = ['name', ['savings', 100.00]]
>>> hubby = person
>>> import copy
>>> wifey = copy.deepcopy(person)
>>> [id(x) for x in person, hubby, wifey]
[12242056, 12242056, 12224232]
>>> hubby[0] = 'joe'
>>> wifey[0] = 'jane'
>>> hubby, wifey
(['joe', ['savings', 100.0]], ['jane', ['savings', 100.0]])
>>> hubby[1][1] = 50.00
>>> hubby, wifey
(['joe', ['savings', 50.0]], ['jane', ['savings', 100.0]])
>>> [id(x) for x in hubby]
[12191712, 11826280]
>>> [id(x) for x in wifey]
[12114080, 12224792]

对于拷贝有几点要注意：

  • 非容器类型（比如数字、字符串和其他“原子”类型的对象，像代码，类型和xrange 对象等）没有被拷贝一说，浅拷贝是通过完全切片操作来实现的
  • 如果元组变量只包含原子类型对象，对它的深拷贝将不会进行
