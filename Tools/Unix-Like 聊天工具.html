身边的朋友一直使用irssi，今天我也试用了下，不错，很简洁很高效，唯一不足的是没有消息自动提示，不过可以写扩展，改天我写一个放上来。还有就是irssi本身只是一个irc的聊天客户端，需要安装bitlee来支持msn、Yahoo、XMPP等协议。安装很简单，如果你的系统和我一样支持apt-get包管理系统，那么就是：sudo apt-get install -y irssi bitlee

安装后直接sudo /etc/init.d/bitlee start启动bitlee服务，然后你就可以irssi进入irssi聊天客户端了。进去后可以连接一个本机的irc服务（作为bitlee的其他协议通讯的网关）/connect localhost，然后你可以通过/window 2切换到bitlee这个窗口中，然后需要通过register命令注册一个bitlee的统一账号，以后只要登录这个，就可以把绑定的IM账号都登录了register 密码。最后就可以通过account add 协议名 账号 密码来增加你的账号。退出irssi的命令是在任意窗口输入：/quit，你再登录irrsi就只要连接（/connect localhost）后跳到第2个窗口，就可以通过identify 密码（前面你register密码注册的）就可以登录所有你的账号了。具体细节可以用help命令查询。

这里还顺便推荐一个Unix工具，screen，具体我就不介绍了，对于办公都在服务器上的同学特别靠谱。这里有一篇IBM的文章，可以好好读一下，写得很详细了。linux 技巧：使用 screen 管理你的远程会话，还有一篇是screen的在线手册，可以一看。

对于screen有一个小技巧，就是默认screen是不支持256色的，这样让我的终端和vim配色都看得难受。其实解决也很简单，在你的家目录的.screenrc文件里加入一行就行。termcapinfo xterm 'Co#256:AB=\E[48;5;%dm:AF=\E[38;5;%dm' # 开启 256 色模式
